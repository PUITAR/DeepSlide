A general-purpose Beamer template using the Berkeley theme, dolphin color scheme, and the infolines outer theme. It includes extensive TikZ utilities (parchment frame macros and flowchart styles) and configured code listings, making it well-suited for technical talks that mix diagrams and code. The deck begins with a title slide, organizes content cleanly, and concludes with a references frame, providing a professional structure for academic or engineering presentations.
