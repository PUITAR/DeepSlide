From Artifacts to Presentation Delivery

DeepSlide is not a tool that “quickly makes a PPT for you”, but a delivery-first, human-in-the-loop system that produces a full presentation package: narrative, slides, interactive preview, and rehearsal insights.

GitHub YouTube Demo Bilibili Demo README

Key claim

A high-quality talk is not mainly determined by whether static slides “look nice”. What matters more is whether information is organized and delivered under audience cognition and attention constraints: narrative coherence, timing and pacing control, attention guidance, and rehearsal readiness.

Four-stage pipeline

Stages

  • Requirement clarification & narrative proposals
  • Logical-chain editing & evidence-grounded generation
  • Interactive enhancement & attention control
  • Rehearsal & evaluation

What you get

  • Time-budgeted logical-chain candidates with node-level editing
  • Co-delivered slides and script
  • In-talk attention strategies (image focus, diagrams, tables)
  • Rehearsal guidance and simulated audience Q&A
