# DeepSlide Streamlit Viewer

基于Streamlit的PDF和演讲稿同步展示工具。

## 功能特点

- 📄 **PDF文件上传和渲染**：支持PDF文件上传，自动转换为高质量图片
- 📝 **演讲稿文本解析**：按`<next>`标签智能分割演讲稿
- 🔄 **同步导航**：PDF页面与演讲稿段落智能映射
- 🎨 **深色主题UI**：现代化的深色界面设计
- 📱 **响应式布局**：适配不同屏幕尺寸
- ⚡ **高性能渲染**：优化的PDF处理和图片渲染

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
streamlit run app.py
```

## 文件结构

```
streamlit_viewer/
├── app.py              # 主应用文件
├── requirements.txt    # 依赖包
├── README.md          # 说明文档
└── sample_files/      # 示例文件（可选）
    ├── sample.pdf
    └── sample.txt
```

## 使用方法

1. **上传文件**：在侧边栏上传PDF文件和对应的演讲稿文本文件
2. **使用示例**：点击"使用示例文件"按钮体验功能
3. **导航控制**：
   - 使用"上一页"/"下一页"按钮
   - 拖动页面滑块
   - 使用键盘方向键（可选）

## 演讲稿格式

演讲稿文本中使用`<next>`标签来标记段落分割：

```
这是第一页的演讲内容<next>
这是第二页的演讲内容<next>
这是第三页的演讲内容
```

## 技术栈

- **Streamlit**：Web应用框架
- **PyMuPDF**：PDF处理和渲染
- **Pillow**：图像处理
- **Python 3.8+**：编程语言
