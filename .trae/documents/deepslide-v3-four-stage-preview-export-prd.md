## 1. Product Overview
DeepSlide v3 是一个将论文 LaTeX 工程快速转成可编辑演示稿的桌面端工作台。
你将按「上传解析 → 需求对话 → 逻辑链编辑 → 预览导出」四阶段完成从论文到可交付初稿的闭环。

## 2. Core Features

### 2.1 User Roles
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 使用者 | 无需注册 | 上传论文工程、对话确认需求、编辑逻辑链、生成预览、导出产物、（可选）进入源码编辑与编译预览 |

### 2.2 Feature Module
产品最小可用形态包含以下页面：
1. **上传与解析页**：压缩包上传、格式校验、解析进度、解析摘要、进入下一阶段。
2. **需求对话页（阶段二）**：对话收集需求、确认生成、无反馈/无反应排查与重试。
3. **逻辑链编辑页**：节点列表编辑、顺序调整、节点详情编辑、引用连线、确认生成预览。
4. **预览与导出页（阶段四工作台）**：生成状态、幻灯预览、导出选项、（可选）切换到源码编辑与编译预览。

## 3. Core Process
你的一条典型流程：
1) 在「上传与解析」上传论文 LaTeX 工程，系统解压并定位 main.tex、抽取摘要与结构。
2) 在「需求对话」用自然语言补齐受众/时长/重点/风格；你点击确认后系统生成逻辑链；若出现无反馈，你按状态条提示进行重试/重连/复制诊断信息。
3) 在「逻辑链编辑」调整顺序、节点摘要与引用连线，确认后生成可预览的初稿。
4) 在「预览与导出」查看幻灯预览并导出；如需更改，可回退到阶段二/三修改并重新生成预览；必要时切换到编辑模式做精修与编译。
