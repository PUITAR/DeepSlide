## 1. Product Overview

DeepSlide v3（React + FastAPI）用于将论文 LaTeX 工程快速转成可编辑的演示文稿工作台。
本阶段对「上传压缩包 / 需求对话 / 逻辑链编辑」三处 UI 做精简现代化改造，并补齐第一阶段语音输入（语音转文字）能力。

## 2. Core Features

### 2.1 User Roles

| 角色  | 注册方式 | 核心权限                                       |
| --- | ---- | ------------------------------------------ |
| 使用者 | 无需注册 | 上传论文工程压缩包、与助手对话收集需求、编辑逻辑链并生成初稿、在工作台编辑与编译预览 |

### 2.2 Feature Module

产品最小可用形态包含以下页面：

1. **上传页**：压缩包上传、格式校验、解析进度与结果摘要。
2. **需求对话页**：聊天记录、文本输入、语音输入（录音→转写→可编辑后发送）。
3. **逻辑链编辑页**：节点时间线、拖拽排序、节点编辑（标题/摘要/时长）、引用连线管理、确认生成。
4. **工作台页**：文件树、源码编辑、编译/预览、AI 指令输入。

## 3. Core Process

你的一条典型流程：

1. 在上传页上传论文 LaTeX 工程压缩包，系统解压并定位 main.tex、抽取 abstract 与章节结构。
2. 进入需求对话页，你通过文本或语音转写输入，逐步补齐 audience/duration/focus/style；当你确认需求后系统自动生成初始逻辑链。
3. 在逻辑链编辑页，你拖拽调整顺序、微调节点标题/摘要/时长，并补充或删除 reference 连线；确认后进入工作台。
4. 在工作台页你编辑 LaTeX 内容并编译，查看预览并继续用 AI 指令迭代。
